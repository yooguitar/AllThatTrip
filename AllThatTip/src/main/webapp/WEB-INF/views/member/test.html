정규표현식

정규표현식(Regular Expression : REGEX / REGEXP)이란,
특정 패턴을 가진 문자열을 찾거나 또는 변경할 때 사용하는 형식 언어
정규표현식을 사용하면 문자열에 대해서 특정 패턴 조건검사를 하거나 또는 변경할 때
복잡한 조건문 / 반복문 을 사용하지 않고 간단하게 처리할 수 있음
자바스크립트 뿐만 아니라 자바/오라클 등 다른 언어들에서도 사용 사능

* 정규표현식 객체 생성 및 정규 표현식과 관련된 메소드들

* 메타문자

메타문자를 이용하여 정규표현식으로 "검색조건으로 삼을 특정 패턴"을 만들 수 있음
문자열이 해당 정규식으로 제시한 패턴에 만족하는지
=> test()통해 검사하거나 replace()을 이용해서 치환

* 플래그문자

/정규표현식/ 뒤에 써줌

종류
i : 대/소문자를 구분하지 않고 비교를 수행
g : 문자열 내의 모든 패턴을 검사하겠다.(전역으로 비교하겠다)
m : 여러 줄 비교를 수행하겠다.
